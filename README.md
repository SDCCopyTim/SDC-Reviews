## Development

### Installing Dependencies

From within the root directory:

```sh
npm install -g webpack
npm install
```

### Database
- For the mongo database, run the seed file from the command lineâ€”while in the root directory, run $ `node database-mg/seedReviews.js`
- For the postgresql database, from the root directory, run $ `psql -d -U [your username] postgres <./database/schema.sql`

### Running the app
1. In a terminal, from the root directory:
```sh
npm run build
npm run start
```
2. Open a browser and navigate to http://localhost:3004
